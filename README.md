# Sitio Web para Restaurante

Este proyecto es un sitio web desarrollado para un restaurante.  
Está construido únicamente con **PHP (sin backend), HTML, CSS y JavaScript**, y funciona como una solución front-end dinámica con componentes reutilizables.

## Tecnologías utilizadas

- PHP para estructurar componentes (header, footer, includes).
- HTML5
- CSS3 (con assets organizados por carpetas)
- JavaScript Vanilla
- Diseño responsive

## Estructura del proyecto

/assets  
 ├── css  
 ├── docs  
 ├── fonts  
 ├── images  
 └── js

/includes  
 ├── header.php  
 └── footer.php

/pages  
 ├── restaurante.php  
 ├── panaderia.php  
 └── eventos.php

index.php

## Características principales

- Página modular (header/footer reutilizables).
- Sección del restaurante, panadería y eventos.
- Optimizado para carga rápida.
- Adaptado para uso en dispositivos móviles.

## Nota sobre imágenes

Las imágenes incluidas pertenecen al restaurante y son parte del sitio publicado públicamente.
